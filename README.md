# PixelFlow
A Processing/Java library for high performance GPU-Computing (GLSL).

# Features

+ FluidSimulation
+ Optical Flow
+ Harris Corner Detection
+ Motion Tracking  /  
+ Image Processing Filters
+ Particle Systems
+ GLSL Tools
+ and more ...
