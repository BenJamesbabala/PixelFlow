PixelFlow

A library by Thomas Diewald for the Processing programming environment.
Last update, 09/07/2016.

PixelFlow is a Processing library for high performance GPU-Computing (GLSL), like Fluid Simulation, Optical Flow, Image processing, Tracking, etc.

FluidSimulation / Optical Flow / Harris Corner Detection / Motion Tracking / Image Processing Filters / Particle Systems / Voronoi / GLSL Tools /

Download

Download PixelFlow version 0.19 (7) in .zip format.

Installation

Unzip and put the extracted PixelFlow folder into the libraries folder of your Processing sketches. Reference and examples are included in the PixelFlow folder.

Keywords. GLSL

Reference. Have a look at the javadoc reference here. A copy of the reference is included in the .zip as well.

Source. The source code of PixelFlow is available at GitHub, and its repository can be browsed here.
